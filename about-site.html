Over Deze Website

Deze portfolio website is ontworpen en gebouwd om mijn vaardigheden als beginnend Software Developer te tonen, gebruikmakend van moderne frontend technologieën zoals Bootstrap 5, HTML5 en CSS3. Het combineert functionele structuur met een strak design.

Design Inspiratie

Voor dit ontwerp ben ik geïnspireerd door de moderne glassmorphism trend in combinatie met een strak, donker thema (dark mode). Door het gebruik van semi-transparante achtergronden met een blur effect (via de CSS property backdrop-filter), creëert de website een gevoel van diepte en professionaliteit zonder dat de inhoud overweldigend wordt.
